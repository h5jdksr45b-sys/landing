O que é esta supervisão?

Esta não é uma aula gravada. Não é um curso teórico.

É supervisão prática, estratégica e aplicada a casos reais.

✔ 2 encontros estratégicos ao vivo (2h30 cada)
✔ Discussão aprofundada de casos reais dos participantes
✔ Material estruturado para aplicação clínica imediata
✔ Grupo fechado ativo por 3 meses
✔ Respostas técnicas em áudio (prazo definido)
✔ Vagas limitadas para manter qualidade técnica

Como funciona o grupo

• Grupo fechado com número reduzido de participantes
• Envio estruturado de casos para análise
• Diretrizes claras de funcionamento
• Participação ativa obrigatória
• Ambiente técnico, objetivo e protegido
